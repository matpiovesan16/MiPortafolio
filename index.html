Sobre Mi.

Desarrollador Web & Estudiante de Ingenieria en Sistemas

Soy un apasionado desarrollador web en constante crecimiento. Como estudiante avanzado de Ingeniería en Sistemas, he encontrado mi pasión en el mundo del desarrollo web. Mi experiencia abarca el uso de JavaScript, React.js y Node.js, y soy habilidoso en el diseño utilizando las librerías Bootstrap y Tailwind CSS para crear experiencias web sorprendentes a demas de tener conocimientos en el lenguaje Java. Lo que me hace destacar es mi capacidad para aprender rápidamente y enfrentar desafíos con entusiasmo. Además, estoy ampliando mis horizontes y actualmente estoy inmerso en el aprendizaje de C# y .NET para diversificar mis habilidades.

Mis Proyectos

Build Growth

Diseñe y desarrollé la pagina de mi agencia digital, en la cual nos dedicamos a escalar las ventas de nuestros clientes, mediantes marketing digital, seo y el desarrollo de paginas web, donde logramos la captacion de leads

Admin Series

Desarrollé un administrador de series con el lenguaje Java usando jframe para las vistas y usando como patron el MVC, el cual se conecta a una base de datos, el cual permite agregar, borrar, actualizar y buscar.

Menu lo del Cordero

Desarrollé un innovador menú digital para el bar 'Lo del Cordero' utilizando React.js. Mi rol central fue la codificación, creando una experiencia interactiva y atractiva para los clientes. Este proyecto permite a los clientes explorar el menú de manera eficiente y ha mejorado la experiencia general en el bar. El desarrollo se hizo exclusivamente para dispositivos móviles, ya que los clientes van a usar la app escaneando un código QR, colocados en cada una de las mesas del local

BlackFish Tattoo

Mi proyecto final en la diplomatura de Programador Full Stack fue la creación de un sitio web completo para el estudio de tatuajes 'Blackfish Tattoo'. Desarrollé el frontend utilizando React.js y el backend con Node.js. Esta plataforma digital proporciona a los clientes una experiencia interactiva para explorar el trabajo del estudio, programar citas y obtener información relevante. Fue un desafío emocionante que me permitió aplicar mis habilidades y crear una solución valiosa para el estudio y sus clientes.

The Rick and Morty WiKi

Desarrollé una aplicación que se conecta al API de 'Rick and Morty' para buscar y mostrar información detallada de los personajes de esta popular serie. Los usuarios pueden explorar y buscar sus personajes favoritos de una manera fácil y divertida. Esta aplicación no solo demuestra mi habilidad para trabajar con APIs, sino también mi pasión por crear experiencias interactivas

To DO List

Desarrollé una aplicación de lista de tareas utilizando Vite.js. Esta aplicación permite a los usuarios gestionar sus tareas de manera eficiente, añadiendo, editando y marcando tareas como completadas. Con una interfaz rápida y amigable, esta aplicación demuestra mi capacidad para crear aplicaciones web modernas y receptivas utilizando Vite.js, brindando una experiencia fluida para la gestión de tareas diarias.
