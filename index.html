Mis Proyectos

Build Growth

Concebí y ejecuté la creación de la plataforma para nuestra agencia digital, enfocada en potenciar las ventas de nuestros clientes mediante estrategias de marketing digital, SEO y el desarrollo de páginas web. Nuestra destacada competencia se evidencia en la efectiva captación de leads que conseguimos.

Admin Series

Diseñé un sistema de gestión de series mediante el framework Spring Boot, empleando Hibernate como mapeador objeto-relacional (ORM) y Maven como gestor de dependencias. Para la interfaz de usuario, opté por el motor de plantillas Thymeleaf, mientras que para la presentación y estilización, implementé Bootstrap. La API desarrollada permite llevar a cabo operaciones fundamentales como la adición, eliminación y actualización de registros.

Menu lo del Cordero

Desarrollé un menú digital para el bar 'Lo del Cordero' utilizando React.js. Mi rol central fue la codificación, creando una experiencia interactiva para los clientes. Este proyecto permite a los clientes explorar el menú de manera eficiente y ha mejorado la experiencia general en el bar. El desarrollo se hizo exclusivamente para dispositivos móviles, ya que los clientes van a usar la app escaneando un código QR, colocados en cada una de las mesas del local

The Rick and Morty WiKi

Desarrollo de una aplicación que se enlaza con el API de 'Rick and Morty' para recuperar y presentar información detallada acerca de los personajes, locaciones y capitulos. Los usuarios tienen la capacidad de explorar y buscar sus personajes predilectos de manera sencilla mediantes diferentes filtros.

To DO List

Desarrollé una aplicación de lista de tareas utilizando Vite.js. Esta aplicación permite a los usuarios gestionar sus tareas de manera eficiente, añadiendo, editando y marcando tareas como completadas. Con una interfaz rápida y amigable, esta aplicación demuestra mi capacidad para crear aplicaciones web modernas y receptivas utilizando Vite.js, brindando una experiencia fluida para la gestión de tareas diarias.
